The Sentient Earth Oracle bridges this gap by recognizing a profound truth: our emotions and Earth's conditions are intimately connected. When air quality drops, anxiety rises. When storms approach, restlessness follows. When nature flourishes, so do we.

## Brand Style Guide

### Color Palette

**Primary Colors:**
- **Cosmic Purple**: #9B59B6 (Mystical wisdom)
- **Earth Blue**: #3498DB (Environmental connection)
- **Aurora Green**: #2ECC71 (Growth and healing)

**Emotion-Reactive Colors:**
- Joy: #FFD700 (Golden warmth)
- Calm: #4ECDC4 (Tranquil teal)
- Anxiety: #FF6B6B (Gentle coral)
- Hope: #A8E6CF (Soft mint)
- Contemplative: #BB8FCE (Lavender wisdom)

**Neutral Colors:**
- **Deep Space**: #0F0F23 (Primary background)
- **Stardust**: #1A1A2E (Secondary background)
- **Moonlight**: #FFFFFF (Primary text)
- **Nebula**: #9CA3AF (Secondary text)

### Typography

**Primary Font:** Inter (Clean, modern, accessible)
- Headlines: Inter Bold 700
- Body: Inter Regular 400
- Captions: Inter Medium 500

**Accent Font:** Playfair Display (Mystical, elegant)
- Oracle quotes and poetic elements only

### Visual Elements

**Glassmorphism:** Translucent cards with backdrop blur
**Particles:** Floating cosmic elements
**Gradients:** Smooth color transitions
**Shadows:** Soft, glowing effects
**Icons:** Cosmic and nature-themed emojis
